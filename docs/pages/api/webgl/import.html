引入和初始化
0.2.0 新增

Web端 

import { WebGL } from 'vislite'

// 获取画笔
let painter = new WebGL(el)

其中el是画布的挂载点，比如可以是一个DIV节点。

配置

创建画笔的时候，你还可以传递一个参数来进行更精细的控制： 

new WebGL(el, {

    // 布尔类型，默认true，表示是否支持区域管理（v0.4.1开始支持）
    "region": boolean,

    // 字符串类型，设置画布缩放模式，可选（v0.8.0开始支持）
    "mode"：string
})
